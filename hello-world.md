## Что это за блог?

Этот блог создан с использованием чистых HTML, CSS и JavaScript.
* **HTML** для структуры страницы.
* **CSS** для стилей.
* **JavaScript** для динамической загрузки содержимого статей.
* **Markdown** для написания самих статей!
